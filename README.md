WhatsApp Bot com Python
Sobre o Projeto

Este projeto é uma automação desenvolvida em Python para enviar mensagens pelo WhatsApp Web a partir de uma planilha Excel.

O sistema lê os dados dos contatos em um arquivo .xlsx, gera mensagens personalizadas e realiza o envio automaticamente utilizando o navegador.

Funcionalidades
Leitura de dados através de planilhas Excel.
Personalização automática das mensagens.
Abertura automática do WhatsApp Web.
Envio de mensagens para múltiplos contatos.
Registro de erros durante a execução.
Tecnologias Utilizadas
Python 3
OpenPyXL
PyAutoGUI
WhatsApp Web
Formato da Planilha

A planilha deve conter as seguintes colunas:

Nome	Telefone	Vencimento
João	5511999999999	10/06/2026
Maria	5511988888888	15/06/2026
Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos de programação em Python, automação de tarefas, manipulação de arquivos Excel e utilização de bibliotecas externas.

Observações

O uso excessivo de automações pode resultar em limitações ou bloqueios por parte do WhatsApp. Utilize o projeto com responsabilidade.
